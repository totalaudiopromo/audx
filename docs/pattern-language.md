# Pattern Language

## Goal

Terse, composable, live-patchable. No PhD required.

## Examples

```
kick 4/4                  # four-on-the-floor kick drum
hh 12x8                   # 1/16 hihats, 12 steps per 8-beat bar
snare 2/8                 # snare on beats 2 and 6 (backbeat)
bass synth C2 | arp 1/8   # synth bass playing C2, arpeggiated 1/8 notes
```
