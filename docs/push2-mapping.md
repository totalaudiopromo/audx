# Push 2 Mapping

## Pads (8×8 grid)

- Pads light up with step status in pattern grid
- Velocity-sensitive
- Pad 0–63 mapped to MIDI notes 36–99 (GM drum mapping)

## Encoders (8)

- Encoder 1: Selected track level
- Encoder 2: Pan
- Encoder 3: Send 1 level
- Encoder 4: Send 2 level
- Encoder 5–8: FX parameters

## Display

Top: mixer levels
Middle: pattern grid
Bottom: transport (BPM, bar/beat)

## Buttons

Play / Stop / Record
Scene 1–8: switch pattern scene
Shift + pad: select track
